# Dan's garden

Dan's digital garden.

Built using Quartz v4: <https://quartz.jzhao.xyz>

## Running locally

```sh
npx quartz build --serve
```

<https://quartz.jzhao.xyz/build>

Or just run:

```sh
npm start
```

## Upgrading Quartz

```sh
npx quartz update
```

<https://quartz.jzhao.xyz/upgrading>
